# DotHyphen

A basic ASCII to [Morse](https://en.wikipedia.org/wiki/Morse_code) translator.

## Usage

```rust
use dothyphen::translate;

let translated_string = translate("Hello World");
println!("{}", translated_string); // Should print .... . .-.. .-.. --- / .-- --- .-. .-.. -..
```
